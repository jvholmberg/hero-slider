REQUIREMENTS

-CSS-reset
-jQuery


INSTALL

1. Add hero-slider folder to your project
2. Add slide-imageSlides to hero-slider/img/
3. Create following DOM-structure

    <div class='hero-slider'>
        <ul class='hero-slides'>
            <li class='hero-slide slide-0'></li>
            <li class='hero-slide slide-1'></li>
            <li class='hero-slide slide-2'></li>
            <li class='hero-slide slide-0 last-hero'></li>
        </ul>
        <div class='hero-overlay'>
            <div class="overlay-browse"></div>
            <div class="overlay-radio"></div>
        </div>
    </div>

    The last element in .hero-slides NEED to share class-name with first
    element. You may add as many imageSlides as you may please.

3. Add the filenames to imageSlides(array) under Configuration in slider.js
4. Config the speed and sleep to you liking

ENJOY :)

*/
